To start with learning to rank, we collected judgments. Judgments grade how good a document is for the query. "Rambo" is an exact match for the query "Rambo" so we grade it a 4. "First Daughter" has nothing to do with Rambo, we grade it a 0.

Each keyword is given a query id (qid) at the top of the file. Scroll down and you'll see each qid given a grade for a document, in the form 
(grade) qid:(qid) # (document identifier)
(This file format is adapted from SVMRank.)
